This guide assumes you have Ruby running on your machine.
Each line is a seperate command.

OCaml Installation:

  • Please read carefully the instructions before excuting the commands:

    1. Get the brew package manager:
      • Linux/C9.io:
        • ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Linuxbrew/install/master/install)"
        PATH="$HOME/.linuxbrew/bin:$PATH"
        echo 'export PATH="$HOME/.linuxbrew/bin:$PATH"' >>~/.bash_profile
      • Mac:
        • ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
    2. Install Ocaml(4-6 min):
      3. brew install ocaml
    3. Install OPAM(1-2 min):
      4. brew install opam
    4. Initialize OPAM
      5. opam init
    5. Configure your .bashsrc (THESE ARE BACK TICKS):
      6. eval `opam config env`
    6. Install Ocamlfind and when prompted hit 'y' (without the quotes):
      7. opam install ocamlfind
    7. Install oUnit and when prompted hit 'y' (without the quotes):
      8. opam install ounit
    8. Navigate into the project folder:
      9. cd project_directory
    9. You are ready to hit make:
      10. make
    10. Now run the tests:
      11. ./public.native
